# Tourney Time

Tournament Bracket Research App with AI-powered analysis.

## Features

- Discover tournament brackets using AI
- Research each matchup with citations from trusted sources
- Get AI-powered winner recommendations with confidence levels
- Track your predictions through the bracket
- View game dates and times for each matchup

## Tech Stack

- React + TypeScript frontend
- Express.js backend
- Perplexity AI for research
- Tailwind CSS + shadcn/ui for styling
